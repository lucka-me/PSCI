Potori | Intro & Privacy 介绍 & 隐私

Potori is a web application to gather and visualize your Ingress Portal nominations. Potori will obtain your mails related to Portal nominations from your Gmail inbox, so you must login with your account to use it. Potori是一个可以收集您的Ingress能量塔提名并可视化的网页应用。Potori需要从您的Gmail邮箱中获取与能量塔提名相关的邮件，因此您必须登录您的账号才能使用。

Permission 权限

Potori requires the following permissions: Potori需要以下权限：

Permission 权限 Describe 描述
Gmail: Read Only Gmail：只读
REQUIRED 必须 		This is the core permission on which Potori is dependent. Potori will obtain mails related to Ingress Portal nomination, then extract and display informations from them.
The mail IDs will be saved in data file if you save the data file in your computer or your Google Drive. Except that, all of your mail data will stay in your browser page. 这是Potori运行所需的核心权限。Potori将获取与能量塔提名相关的邮件并提取、展示其中的信息。
如果您将数据文件保存在电脑或您的Google Drive中，邮件ID将被存储在数据文件中。除此之外您的所有邮件数据将停留在浏览器页面内。
Google Drive: Files Google Drive：文件
RECOMMENDED 建议 		Potori could upload your nomination data to your Google Drive manually, and download the data before fetching your mails.
By using the data file, Potori will not obtain and process all mails every time, which could reduce the network traffic and time useage.
The file will be storaged in a private app folder where only Potori can access. Potori允许您将提名数据手动上传至您的Google Drive，下次获取邮件前将下载这些数据。
通过使用数据文件，Potori可以不需要每次都获取和处理所有邮件，这能有效减少网络流量和耗时。
这些文件将被存储在一个私有的应用文件夹中，只有Potori能够访问。

Workflow 工作流程

This part will describe how Potori works.
For those mails related to Portal nominations, the subjects and contents are generated from several templates, some keywords are used to filter mails and extract informations. 这部分将阐述Potori如何运作。
与能量塔提名相关的邮件，它们的主题和内容由一些模版产生，一些关键词可用于筛选邮件和提取信息。

  1. Open or download the nomination data file
    If you have saved a nomination data file locally, you can open it to review your nominations without login.
    After login, Potori will try to download nomination data from your Google Drive if you have uploaded before.     打开或下载提名数据文件
    如果您在本地保存了数据文件，您可以直接打开它并查看您的提名而无需登录。
    如果您曾经上传过数据文件，Potori将在登录后从您的Google Drive下载文件。
  2. Obtain the target mails with filters
    Potori will obtain mails with the following filters:     通过筛选器获取目标邮件
    Potori将利用以下筛选器获取邮件：
    Mails 邮件 Filters 筛选器
    Confirmation 提名确认 from:ingress-support@nianticlabs.com Portal submission confirmation -edit -photo
    from:nominations@portals.ingress.com subject:("能量塔提交確認" OR "Portal申請の受領確認" OR "Portal submission confirmation") -AP
    Acceptance 提名接受 from:ingress-support@nianticlabs.com Portal review complete now available -edit -photo
    from:nominations@portals.ingress.com Intel Map AP
    Rejection 提名拒绝 from:ingress-support@nianticlabs.com Portal review complete reviewed -edit -photo
    from:nominations@portals.ingress.com subject:("能量塔審查完畢" OR "Portal審査の完了" OR "Portal review complete") -AP

    Tips: You can also create rules with these filter to label your mails automatically in Gmail. 贴士：您可以使用这些筛选器在您的Gmail中创建规则以自动分类您的邮件。
  3. Extract informations from the mails
    The contents will be used to extract information of the nomination, browse Data Usage for more details.     从邮件中提取信息
    邮件内容将被用于提取提名信息，详情请见数据使用部分。
    As for the rejection mails, there are several templates for each rejection reason, and each one colud be identified by matching the keywords: 对于拒绝邮件，有几种不同的模版分别对应不同的拒绝原因，可以通过以下关键词来识别：
    Reason 原因 Keywords 关键词
    Duplicated 重复 duplicate of either an existing Portal
    Too Close 过近 too close to an existing Portal
    能量塔過近
    Undeclared 未指明 (Match neither of the keywords above) （不匹配以上任何关键词）
  4. Visualize
    Potori will display the information of nominations in cards, and display data and stastistics in the Dashboard view.
    The status, result time and rejection reason are editable, click the cards to open the Details dialog and edit.     可视化
    Potori会将提名信息展现在卡片中，并在仪表盘视图中展示数据和统计信息。
    提名的状态、结果时间和拒绝理由可以在点击卡片打开的详情对话框中编辑。
  5. Import
    Potori can import data from Wayfarer API and update the title and location of nominations. Click Import from menu to open the Import dialog, paste the JSON code from Wayfarer API and click Import.     导入
    Potori可以导入Wayfarer API中的数据以更新提名的名称和坐标。在菜单中点击Import打开导入对话框，将Wayfarer API的JSON代码粘贴并点击导入即可。
  6. Save or upload the nomination data file
    You can save the nomination data to your computer by clicking the Save File from menu, a file named Potori.json will be downloaded.
    The nomination data will NOT be uploaded to your Google Drive automatically. If you have login, you can upload the data by clicking the Upload File from menu.     保存或上传提名数据文件
    您可以通过点击菜单中的Save File来将提名数据保存在本地，浏览器将会下载一份名为Potori.json的文件。
    提名数据不会被自动上传至Google Drive。如果您已经登录，可以点击菜单中的Upload File来上传文件。

That's how Potori gets information of your Portal nominations from your inbox. Ingress Prime uses localized templates, which makes it difficult to find the corrent keywords, We are working on it. 以上介绍了Potori如何从您的收件箱中获取您的能量塔提名信息。Ingress Prime使用了本地化的模版，这给确定关键词带来了挑战，我们正在尝试解决这个问题。

Privacy 隐私

Briefly, Potori would not collect any of your data. All of your data will be handled inside your browser, and deleted when you close the page. You can save your data in your own Google Drive (where only Potori can access) or locally, they are totally under your control. 简而言之，Potori不会收集您的任何数据。您的所有数据将在您的浏览器内进行处理，当您关闭页面时，数据也会被清除。您可以将数据保存在您的Google Drive中（只有Potori能够访问的位置）或本地，它们完全处于您的控制之下。

I'm trying to protect your privacy with my knowledge, if you have any suggestion, please feel free to contact us or open an issue. 在下尽力设法保护您的隐私，如果您有任何建议，欢迎联系我们或提出issue。

Data Usage 数据使用

  • Mails related to Portal nomination 与申请Portal有关的邮件
    Potori will obtain and analyze confirmation, acceptance and rejection mails only. Potori仅会获取和分析确认邮件、通过邮件和拒绝邮件。
  • Content of the mail 邮件的内容
    Content 内容 Useage 被用于
    Mail ID 邮件ID Avoid obtaining and processing the same mail every time, if you save or upload the data file. 如果您保存或上传数据文件，邮件ID可用于避免重复获取和处理相同的邮件。
    Subject 标题 Extract title of Portal. 获取能量塔名称。
    Timestamp 时间戳 Regarded as the confirmation / acceptance / rejection time of the nomination. 作为提名的确认/接受/拒绝时间。
    Photo URL 照片地址 Show photo in the card.
    Generate a short UID as the ID of Portal and identify them in mails.
    [FULL VERSION ONLY] The short UID will be used to query Brainstorming database and generate link to Brainstorming watermeter.     在卡片中显示照片。
    生成一个短UID，作为能量塔ID以及在邮件中识别它们。
    [仅限完整版] 短UID将被用于查询Brainstorming数据库以及生成Brainstorming水表墙的链接。
    Intel Map URL Intel Map地址 Extract the coordinate and display a marker on the map. 获取坐标并在地图中显示标记。
    Mail Body 邮件体 Extract the rejection reason. 获取拒绝原因。

About the Data File 关于数据文件

To reduce the network traffic and time useage, Potori provides a feature which allows you to save the data file and upload it to a private app folder in your own Google Drive where only Potori could access. The informations contained in the file are described above. 为了减少网络使用和处理时间，Potori提供了一个功能让您能够保存数据文件或将其上传至您的Google Drive中的一个仅限Potori访问的私有应用文件夹。文件中包含的信息如前文所述。
