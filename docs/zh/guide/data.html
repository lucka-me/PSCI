# 导入和导出

如果您在本地保存了数据文件，您可以直接打开它并查看您的提名而无需登录。

如果您曾经上传过数据文件，Potori将在登录后从您的Google Drive下载文件。

您可以通过点击菜单中的Save File来将提名数据和Brainstorming数据保存在本地，浏览器将会下载分别名为potori.json和bsdata.json的文件。

提名数据不会被自动上传至Google Drive。如果您已经登录，可以点击菜单中的Upload File来上传文件。

# 导入Wayfarer JSON

Potori可以导入Wayfarer APIopen in new window中的数据以更新提名的名称和坐标。在菜单中点击Import打开导入对话框，将Wayfarer API的JSON代码粘贴并点击Import即可。
