# 贡献

向Potori添砖加瓦是非常简单的，您可以（但不限于）：

  • 提供邮件样本以完善邮件筛选器和新的拒绝原因的关键词
  • 改善Webpack打包
  • 报告或修正问题
  • 建议，例如新的卡片或功能
